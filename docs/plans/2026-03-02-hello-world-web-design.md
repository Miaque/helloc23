# Hello World Web 应用设计

## 概述

一个简单的 C++ web 应用，使用 Drogon 框架实现，访问主页时返回 "hello world"。

## 技术栈

| 组件 | 选择 |
|------|------|
| 框架 | Drogon |
| C++ 标准 | C++20 |
| 构建系统 | CMake |

## 架构

```
┌─────────────────────────────────────┐
│         HTTP 客户端 (浏览器)          │
└──────────────┬──────────────────────┘
               │ GET /
               ↓
┌─────────────────────────────────────┐
│      Drogon HTTP 服务器             │
│  - 监听端口 (默认 8080)            │
│  - 路由处理                        │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│   HelloController 处理器            │
│  - 处理 GET / 请求                 │
│  - 返回 "hello world" 响应         │
└─────────────────────────────────────┘
```

## 组件

1. **HelloController** - 控制器类，处理 HTTP 请求
2. **main.cpp** - 应用入口，启动 Drogon 服务器
3. **CMakeLists.txt** - 更新以包含 Drogon 依赖

## API 接口

| 端点 | 方法 | 响应 |
|------|------|------|
| `/` | GET | `hello world` |

## 数据流

```
客户端 → HTTP GET / → Drogon 路由 → HelloController::hello() → 返回 "hello world" → 客户端
```
